# Urban Routes - Testes Manuais

Projeto de testes manuais realizado durante bootcamp de QA.

## Objetivo
Validar funcionalidades do aplicativo Urban Routes, garantindo que o sistema funcione corretamente e identificando possíveis falhas.

## Atividades realizadas
- Criação de casos de teste
- Execução de testes manuais
- Registro de bugs
- Documentação de evidências
