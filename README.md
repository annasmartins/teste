# Urban Routes - Regression Testing

Projeto de teste de regressão manual no aplicativo Urban Routes, com execução de casos de teste e registro de bugs.

## Sobre o projeto
O Urban Routes é um aplicativo de rotas que calcula tempo e custo de viagens para diferentes meios de transporte. A interface permite informar origem e destino e visualizar opções como caminhada, táxi, bicicleta, scooter e carro.

Neste projeto, o objetivo foi validar o comportamento da aplicação por meio da execução de casos de teste preparados pela equipe de QA, registrando os resultados e documentando os bugs encontrados.

## Objetivo
Executar testes de regressão no aplicativo Urban Routes para verificar se as funcionalidades principais continuam funcionando corretamente após mudanças no sistema.

## Atividades realizadas
- Execução de casos de teste manuais
- Registro de status dos testes
- Identificação de falhas
- Criação de relatórios de bugs
- Análise de comportamento esperado versus comportamento real

## Escopo analisado
- Inserção de endereços de origem e destino
- Seleção de modos e tipos de transporte
- Cálculo de tempo e custo da viagem
- Comportamento geral da interface

## Arquivos deste repositório
- `test-execution-summary.md` → resumo da execução dos testes
- `test-cases.md` → exemplos de casos de teste
- `bug-reports.md` → exemplos de bugs documentados
